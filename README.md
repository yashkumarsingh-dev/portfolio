# My Portfolio Website

This is my personal portfolio website built using **React** and **Tailwind CSS**. The website showcases my skills, projects, hobbies, and some basic information about me. It also features a **typed.js** animation to display a dynamic introduction message.

## Features

- **Responsive Layout**: The design is mobile-friendly and adapts to various screen sizes.
- **Typed.js Animation**: Dynamically displays messages like "I am Yash" and "I am a developer."
- **Tailwind CSS**: Utility-first CSS framework for clean and fast styling.
- **Custom Background**: The website uses a full-screen background image.

## Technologies Used

- **React**: A JavaScript library for building user interfaces.
- **Tailwind CSS**: A utility-first CSS framework for styling the website.
- **Typed.js**: A JavaScript library for creating typing animations.
- **GitHub Pages**: For hosting the website.

## How to Run Locally

1. Clone the repository to your local machine:

   ```bash
   git clone https://github.com/yashkumarsingh-dev/portfolio.git

2. Navigate to the project directory:
    ```bash
    cd portfolio
    ```

3. Install dependencies:
    ```bash
    npm install
    ```
4. Start the development server:
    ```bash
    npm start
    ```
5. Open your browser and go to `http://localhost:3000` to view the website.
## Deployment
